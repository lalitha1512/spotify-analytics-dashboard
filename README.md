# spotify-analytics-dashboard
Spotify Analytics Dashboard focusing on user behavior and churn analysis using Power BI

## Spotify Analytics Dashboard | User Behavior & Churn Analysis

An interactive Power BI dashboard that analyzes Spotify user behavior, subscription trends, device usage, and churn patterns using a Spotify-themed dark UI.

## Project Overview
This project focuses on understanding how users interact with Spotify by visualizing listening behavior, subscription types, ad exposure, and churn trends.

## Tools Used
- Power BI Desktop
- CSV / Excel Dataset
- Data Visualization & Analytics

## Dashboard Features
- KPI cards for total, active, and churned users
- Subscription analysis by country
- Device usage distribution
- Ads and skip rate analysis
- Churn behavior insights
- Interactive slicers
- Spotify-themed dark UI

## Dataset
The dataset contains user demographics, listening behavior, subscription details, and churn information.

## Key Insights
- Premium users show higher engagement and lower churn
- Higher ad exposure correlates with higher skip rate and churn
- Mobile and Web platforms dominate usage
- User behavior varies significantly across age groups
